The Open Source Audit: [Vlc media player]


📌 Project Overview
This project is a comprehensive audit of [Vlc media player], conducted as a Capstone Project for the Open Source Software (OSS) course. The goal of this audit is to explore the origin, licensing, and systemic impact of the software, alongside practical Linux automation through shell scripting.
